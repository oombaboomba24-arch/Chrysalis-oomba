# GitHub

## Business Overview
GitHub is the world's largest software development platform, owned by Microsoft since 2018 for $7.5B. It hosts 100M+ developers and is the primary surface for AI-assisted development (Copilot). GitHub's enterprise offering (GitHub Enterprise, GitHub Advanced Security) is a major and growing revenue contributor. Identity at GitHub spans user accounts, enterprise SSO via SAML/OIDC, non-human identity (Actions tokens, service accounts), and GitHub's own OAuth apps ecosystem. ([GitHub About](https://github.com/about))
